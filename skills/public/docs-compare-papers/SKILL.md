# Compare Papers Skill

Use this workflow when the user asks for cross-paper comparison.

Deliver:
- comparison dimensions
- shared ideas
- major differences
- strengths or weaknesses
- evidence-backed conclusion

Rules:
- require at least two readable documents
- avoid fake comparisons based only on titles
- cite each comparison claim with document references
